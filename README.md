# Java-Swing-Shopping-Mall

این پروژه یک اپلیکیشن دسکتاپ برای مدیریت فروشگاه و خرید کالا است که با زبان Java و کتابخانه Swing طراحی شده است. برنامه دارای دو سطح دسترسی مدیر (Admin) و مشتری (Customer) می‌باشد.

# ویژگی‌های کلیدی

1. پنل مدیریت (Admin Panel)
مدیریت محصولات: امکان افزودن، ویرایش و حذف محصولات (نام، دسته‌بندی، قیمت، موجودی و توضیحات).

مدیریت تصاویر: امکان انتخاب تصویر برای هر محصول و کپی خودکار آن در پوشه پروژه برای مدیریت بهتر.

مشاهده لیست: نمایش تمام محصولات در یک جدول با قابلیت به‌روزرسانی آنی.

2. پنل مشتری (Customer Panel)
جستجو و فیلتر: قابلیت جستجوی متنی محصولات و فیلتر کردن بر اساس دسته‌بندی (Electronics, Clothing, Food, etc).

پیش‌نمایش تصویر: با انتخاب هر محصول در جدول، تصویر آن در کنار صفحه نمایش داده می‌شود.

سبد خرید (Shopping Cart): امکان افزودن کالا به سبد، مدیریت تعداد و مشاهده قیمت کل.

تسویه حساب (Checkout): بررسی موجودی انبار در هنگام خرید و کسر خودکار از موجودی پس از تایید نهایی.

3. سیستم ذخیره‌سازی (Persistence)
داده‌ها در فایل‌های products.csv و cart.csv ذخیره می‌شوند تا با بستن و باز کردن برنامه، اطلاعات حفظ شوند.

استفاده از الگوی Repository برای جداسازی منطق ذخیره‌سازی از منطق برنامه.

# ساختار فنی کدهای پروژه

این پروژه از اصول Object-Oriented Programming (OOP) به خوبی بهره برده است:

Models: کلاس‌های Product و CartItem برای نگهداری موجودیت‌ها.

User Hierarchy: استفاده از کلاس انتزاعی User و ارث‌بری کلاس‌های Customer و Administrator.

Services: کلاس‌های ProductService و CartService برای مدیریت عملیات منطقی (Business Logic).

GUI (Swing):

استفاده از CardLayout برای جابجایی بدون درز بین صفحات لاگین، ادمین و مشتری.

طراحی ریسپانسیو نسبی با استفاده از GridBagLayout و BorderLayout.

# برخی از پیاده سازی ها

مدیریت تصویر: متد copyFileToImages به خوبی فایل‌ها را با نام منحصر‌بفرد (UUID) کپی می‌کند تا تداخل نام پیش نیاید.

Thread-safe-like logic: استفاده از WindowAdapter برای ذخیره خودکار اطلاعات در فایل هنگام بستن برنامه (Event Handling).
